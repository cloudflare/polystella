# Expressions And Catalog Boundaries

<Callout title={`Hello ${name}`} type="info">

Dynamic expressions should not be translated by evaluating JavaScript. They belong in catalogs, frontmatter, or explicitly annotated static data.

</Callout>

<Callout title={"Static expression string"} type="success">

Even static expression strings are intentionally not part of the first safe boundary.

</Callout>
